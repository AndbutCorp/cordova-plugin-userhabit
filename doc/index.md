# io.userhabit.plugin.cordova

This plugin provides to use Userhabit.

## Installation

    cordova plugin add https://github.com/AndbutCorp/cordova-plugin-userhabit.git

### Supported Platforms

- Android

### Example

    cordova.userhabit.passCheckpoint('CHECKPOINT');
